# B. Sc. In Computing
## Introduction to Object-Oriented Design
### Tutorial 1– Simple class definitions

**Aim**: the aim of this tutorial is to ensure that the student understands how to design a simple class and implement such a class in Java The first exercise focuses on simple get and set methods, the second exercise extends the definition of your class to demonstrate some behaviour.

### Exercise 1 

Design and implement a class in Java that stores details id, name and age about a person. Write 3 contructors (1 with no args, 1 with 1 arg, one with 2 args)  and get/set methods for each attribute.

Write a main procedure that creates 3 instances of Person. Use a different constructor for each instance.

Assign different values to some of the attributes using your set methods.

Print out the details of each person using the get methods.



### Exercise 2

Extend the definition of your class to allocate some behaviour to the class. Implement your design in Java and use a main method to demonstrate an object of the class exhibiting this behaviour.

Your answer should include modification and selection type behaviour.


### Exercise 3:

An Employee is a person. Staff are employees but a Professor is a specific type of employee.
A student is a person. Employees are paid an hourly wage while Professors are paid a monthly salary. Create appropriate classes to represent these relationships.
